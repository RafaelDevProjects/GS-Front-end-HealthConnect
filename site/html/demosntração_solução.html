Demonstração da solução

Edge

A solução proposta consiste em um dispositivo Arduino que utiliza um sensor ultrassônico para medir a distância entre o dispositivo e uma pessoa. Quando a distância é inferior a um limiar predefinido, o Arduino registra a entrada de um paciente. Isso é indicado por LEDs de diferentes cores, representando diferentes estados: vermelho para indicação de entrada, amarelo para uma situação intermediária, e verde para indicar que o paciente está longe do sensor. O projeto também incorpora uma funcionalidade de registrar a data e a hora da entrada de cada paciente, permitindo uma análise temporal do fluxo de pacientes.
